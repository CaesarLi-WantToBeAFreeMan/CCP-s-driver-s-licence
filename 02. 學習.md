# 扣分學習

## 學習類型

| 類型          | 範圍                                    |
| :----------: | :-------------------------------------: |
| `審驗教育`    | **大貨車** 扣 $(0, 12)$ 分，**必須** 學習  |
| `學法減分`    | **機動車** 扣 $(0, 12)$ 分，**可以** 學習  |
| `滿分學習`    | 扣超 `12` 分，**必須** 學習               |

## 滿分學習

* 一個週期 (`12` 個月)，累計計分學習表
* 每七天一個迴圈 (網 `5` 現 `2`)
* 公式：

```cpp
    int studyDay = min(score / 12 * 7, 60);
```

| 累計計分數    | 學習內容                                 |
| :---------: | :-------------------------------------: |
| $[12, 24)$  | 法規 (`科一`)                            |
| $[24, 36)$  | 法規 (`科一`)、道路 (`科三`)               |
| $[36, 60]$  | 法規 (`科一`)、場地 (`科二`)、道路 (`科三`) |

## 學法減分

* `實習期`，**禁止** 參加
* 一個週期 (`12` 個月)，扣分在 $(0, 12)$ 的範圍內
* 最高減 `6` 分
1. `公益` `1` 小時減 `1` 分
2. `現場` `1` 小時減 `2` 分
3. `網上` `0.5` 小時減 `1` 分 (`3` 日內累計 `30` 分鐘)

### 口訣

1. 公益 `111`
2. 現場 `112`
3. 網上 `331`
