# 判刑

## 類型

1. `危險駕駛罪`: 輕傷以下、輕微財產損失
    * `追逐競駛`
    * `醉駕`
2. `交通肇事罪`：有重傷死亡、嚴重財產損失
    * `酒駕`
    * `毒駕`
    * `無牌`
    * `報廢`
    * `肇事逃逸`

### 口訣

1. 交通肇事，選 3 「未」之外的選項

## 刑期

| 狀況             | 刑期              |
| :-------------: | :---------------: |
| 未逃逸           | $(0, 3)$ 年       |
| 死亡後逃逸        | $[3, 7]$ 年       |
| 逃抑致死亡        | $(7, \infty)$ 年  |
| 組織作弊不嚴重    | $(0, 3)$ 年        |
| 組織犯罪嚴重      | $[3, 7]$ 年       |
| 公共交通用暴力    | $(0, 1)$ 年       |
